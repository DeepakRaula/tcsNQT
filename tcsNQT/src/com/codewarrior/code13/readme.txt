Find the Nth Term in a Special Series
Problem Statement:
You are given a number N, which represents the position in a special number series: 1, 1, 2, 3, 4, 9, 8, 27, 16, 81, ...
Your task is to find the Nth term of this series.
Input Format:
A single integer N (1 ≤ N ≤ 100).
Output Format:
Print the Nth term of the series.
Example:
Input: 4
Output: 3
Explanation: The 4th term in the sequence is 3.
Input: 7
Output: 8
Explanation: The 7th term in the sequence is 8.
